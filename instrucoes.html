Controles: Use W ou ↑ para pular e D ou → para andar para frente.

🎯

Objetivo: Colete frutas para ganhar pontos e avançar pelas fases do jogo.

🍓

Frutas: Cada fruta coletada vale +3 pontos. Junte o suficiente para passar de fase!

⚠️

Obstáculos: Evite frutas podres e lixo! Cada colisão tira -2 pontos e acumula penalidade.

❤️

Vidas: Você começa com 3 vidas. A cada 5 colisões, perde 1 vida.

🚀

Fases: O jogo tem 3 fases. A cada fase a velocidade aumenta. Complete todas para vencer!

💀

Game Over: Se perder todas as vidas, o jogo acaba.

🔁

Reiniciar: Pressione R para jogar novamente após perder ou vencer.
